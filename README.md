# 轻量报表后台演示

## 目录结构

- `server.js`：Node/Express 服务，提供配置读取/保存接口和数据接口。
- `config/report-config.json`：报表配置存储文件。
- `public/admin.html`：报表配置后台页面。
- `public/dashboard.html`：报表展示页面。
- `public/data.json`：演示数据。

## 运行方式

1. 进入目录：

```bash
cd "轻量报表后台-demo" || cd light-report-demo
```

2. 安装依赖：

```bash
npm install
```

3. 启动服务：

```bash
npm start
```

4. 打开页面：

- 报表配置后台：`http://localhost:3000/admin.html`
- 报表展示页面：`http://localhost:3000/dashboard.html`

## 功能特性

- **现代化UI设计**：采用渐变背景、毛玻璃效果、圆角设计和动画过渡
- **灵活的报表布局**：支持仪表盘 + 图表 + 表格、仅图表 + 表格、仅仪表盘 + 表格三种展示方式
- **仪表盘指标卡片**：实时计算汇总指标，支持点选同步图表和表格
- **实时监控趋势图**：15分钟颗粒度的时间序列图表，横坐标分时段展示，支持数据缩放
- **统一图表展示**：所有指标在单张图表中展示，便于对比分析
- **智能筛选**：报表维度自动转换为筛选控件，支持多条件过滤，控制趋势图显示内容
- **配置管理**：支持导入/导出配置 JSON，方便模板管理

## 使用方法

1. 在 `admin.html` 中编辑报表配置。
2. 右侧"面板列表"可以上下移动面板、单独配置每个面板的图表类型。
3. 在"仪表盘指标"字段配置想要在顶部卡片展示的指标（留空则不显示仪表盘）。
4. 点击"发布配置"保存到 `config/report-config.json`。
5. 点击"导出配置"可把当前配置保存成 JSON 文件。
6. 点击"导入配置"可从本地 JSON 文件加载一套配置，然后再点击"发布配置"。
7. 在 `dashboard.html` 中刷新页面，默认会自动打开首个报表。
8. 报表标题下方的筛选控件可按维度过滤数据。
9. 仪表盘卡片显示实时指标汇总，点选卡片可过滤图表和表格显示的指标。
10. 图表统一在一张图中展示所有选中的指标趋势。

## 注意

当前演示版使用静态模拟数据，主要用于验证报表配置和前端展示流程。
